# 스터디 개요

## 목적

- 「모던 자바스크립트 Deep Dive」를 함께 읽으며 JS의 동작 원리를 구조적으로 이해하기
- "되니까 쓴다"가 아니라 "왜 이렇게 동작하는지 설명할 수 있다" 상태 만들기

# 진행 방식

## 스터디 전

- 매주 스터디 모임 전까지 해당 주차 범위를 읽고 학습 노트에 정리글 업로드
  - 정리글 미제출 시 벌금 3만원 (스터디 참석 여부 무관)
  - 벌금은 스터디 종료 후 회식비로 사용

## 스터디 당일

- 매주 스터디 모임 때 [마블 룰렛](https://lazygyu.github.io/roulette/)으로 결정된 한명이 정리한 핵심 내용을 발표
- 읽으면서 헷갈렸던 부분, 서로의 정리글에서 궁금했던 점 자유 토론
- 다음주 범위 확인 + 분량 조율이 필요할 경우 조정
